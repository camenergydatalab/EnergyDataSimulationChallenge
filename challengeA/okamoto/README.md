# CHALLENGE A

## 概要

契約アンペア(A)と月の電力使用量(kWh)の入力に応じて、各プラン名と料金のシミュレーションを行うスクリプトです。

## シミュレーション実行方法

```sh
ruby main.rb
```

## データの編集

JSON形式で`/data`のディレクトリに格納しています。プラン追加や編集の際はJSONデータを修正することで対応できるように実装しています。

## テスト

```sh
bundle exec rspec
```

## 補足事項

- 合計電気料金は小数点以下切り捨て
